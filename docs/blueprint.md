# **App Name**: Prolift Fitness

## Core Features:

- Member Management: Admin panel to add, update, and manage member profiles, including personal details and contact information.
- Attendance Tracking: Log member check-ins, manage attendance records and provide summary reports to the admin.
- Payment Management: Track payments, send payment reminders, and manage transaction history.
- Automated Notifications & Reminders: System for automated notifications, reminders for class schedules and payment due dates; also system for sending customized announcements and alerts to individual members, a selected group, or the entire member list. This system can also act as a tool for reasoning if specific pieces of information is needed to perform the correct response.
- Subscription Plan Management: Define various membership plans with different pricing and benefits.
- Sleek Landing Page: Visually appealing and informative landing page to showcase fitness offerings and attract new members.

## Style Guidelines:

- Primary color: Strong blue (#29ABE2) reflecting health and trust.
- Background color: Light gray (#F4F4F4), for a clean and modern look.
- Accent color: Energetic orange (#FF8C00) to highlight key actions.
- Body and headline font: 'Inter', a grotesque sans-serif with a modern look.
- Use minimalist icons related to fitness, members, and payments.
- Clean, grid-based layout for a structured user interface and responsive design for cross-device compatibility.
- Subtle transitions and animations for a polished user experience.
